TROUBLESHOOTING
---------------
  No markers in RViz:
    1. Check ORB-SLAM3 is tracking (move camera)
    2. Start YOLO detection from CV menu
    3. Enable "Semantic Labels" in RViz
    4. Check Fixed Frame is "map"
    5. Run: ./scripts/diagnose_markers.sh
  
  No camera pose TF:
    1. Rebuild: cd ros2_ws && colcon build --packages-select kinect2_slam
    2. Check ORB-SLAM3 tracking status
    3. Verify: ros2 topic echo /orb_slam3/pose
  
  No CV detections:
    1. Start YOLO from CV Pipeline menu
    2. Check: ros2 topic echo /cv_pipeline/results
    3. Verify Kinect is publishing images
  
  ORB-SLAM3 "waiting for image":
    1. Check Kinect is connected: lsusb | grep Kinect
    2. Verify topics: ros2 topic list | grep kinect2
    3. Check image rate: ros2 topic hz /kinect2/hd/image_color

SUPPORTED CV MODELS
-------------------
  YOLO11:
    • Detection (bounding boxes)
    • Segmentation (instance masks)
    • Pose estimation (keypoints)
    • OBB (oriented bounding boxes)
  
  SAM2 / FastSAM:
    • Segmentation masks
    • Point/box/text prompts
  
  InsightFace:
    • Face detection
    • Face recognition
    • Emotion detection
    • Age/gender estimation

PERFORMANCE TIPS
----------------
  • Lower TSDF resolution: voxel_length:=0.08
  • Reduce marker lifetime: marker_lifetime:=10.0
  • Increase confidence threshold: conf_threshold:=0.6
  • Limit depth range: depth_trunc:=3.0
